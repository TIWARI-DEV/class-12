Functions in C

Functions are self-contained blocks of code that perform a specific task. They help in breaking down complex problems into smaller, manageable parts.

Function Syntax

return_type function_name(parameters) {
    // function body
}

Components of a Function

Function Prototype

Declares the function before it's defined, specifying return type, name, and parameters.

int add(int a, int b);

Function Definition

Actual implementation of the function with code.

int add(int a, int b) {
    return a + b;
}

Function Call

Invokes the function to execute its code.

int result = add(5, 3);

Return Statement

Exits the function and returns a value to the caller.

return value;

Passing Arguments

Call by Value

Passes a copy of the argument's value. Changes inside the function don't affect the original.

void increment(int x) {
    x++;
}

Call by Reference

Passes the address of the argument. Changes inside the function affect the original.

void incrementRef(int *x) {
    (*x)++;
}

Recursive Functions

A function that calls itself to solve smaller instances of the same problem.

// Factorial using recursion
int factorial(int n) {
    if (n == 0)
        return 1;
    else
        return n * factorial(n - 1);
}
Recursion Advantages:

Elegant solution for problems with recursive nature (trees, graphs, divide-and-conquer).

Recursion Disadvantages:

Can cause stack overflow for large inputs and may be less efficient than iterative solutions.

Structures & Unions

Structures

User-defined data type that groups variables of different types under a single name.

// Structure syntax
struct Student {
    char name[50];
    int roll;
    float marks;
};

Accessing Structure Members

// Create structure variable
struct Student s1;

// Access members
strcpy(s1.name, "John Doe");
s1.roll = 101;
s1.marks = 85.5;

Nested Structures

struct Address {
    char city[50];
    int pincode;
};

struct Employee {
    char name[50];
    struct Address addr;
};

Unions

Similar to structures but all members share the same memory location.

union Data {
    int i;
    float f;
    char str[20];
};

Structure vs. Union

Feature Structure Union
Memory Allocation Each member has its own memory All members share the same memory
Total Size Sum of all members + padding Size of the largest member
Member Access All members accessible at any time Only one member can be accessed at a time
Use Case Group related data of different types Memory-efficient storage for different data types

Pointers in C

Pointers are variables that store memory addresses. They provide powerful capabilities for dynamic memory management and efficient data manipulation.

Pointer Syntax

int num = 10;
int *ptr; // Pointer declaration
ptr = &num; // Assign address of num to ptr

printf("Value: %d\n", *ptr); // Access value using pointer

Pointers and Functions

// Function using pointers
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

// Calling the function
int x = 5, y = 10;
swap(&x, &y);

Pointers and Arrays

int arr[5] = {1, 2, 3, 4, 5};
int *ptr = arr; // Points to first element

// Access array elements using pointer
for(int i = 0; i < 5; i++) {
    printf("%d ", *(ptr + i));
}

Call by Value vs. Call by Reference

Call by Value Call by Reference
Passes a copy of the value Passes the address of the variable
Original value remains unchanged Original value can be modified
Memory efficient for small data Efficient for large data structures
Simple and safe Requires careful handling to avoid errors

File Handling in C

File handling allows programs to store data persistently and access it later. C provides a comprehensive set of functions for file operations.

File Handling Steps

1. Define File Pointer

FILE *filePtr;

2. Open File

filePtr = fopen("filename.txt", "mode");

3. Perform Operations

Read, write, or append data

4. Close File

fclose(filePtr);

File Opening Modes

Mode Description
"r" Open for reading (file must exist)
"w" Open for writing (creates new or truncates existing)
"a" Open for appending (creates file if doesn't exist)
"r+" Open for both reading and writing
"w+" Open for reading and writing (truncates existing)
"a+" Open for reading and appending

File Handling Functions

fopen() & fclose()

FILE *fopen(const char *filename, const char *mode);
int fclose(FILE *stream);

fprintf() & fscanf()

int fprintf(FILE *stream, const char *format, ...);
int fscanf(FILE *stream, const char *format, ...);

fgets() & fputs()

char *fgets(char *str, int n, FILE *stream);
int fputs(const char *str, FILE *stream);

fread() & fwrite()

size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);
size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);

Random Access Functions

fseek()

Set file position to a specific location

int fseek(FILE *stream, long offset, int whence);

rewind()

Set file position to the beginning

void rewind(FILE *stream);

ftell()

Get current file position

long ftell(FILE *stream);

typedef Keyword

The typedef keyword allows you to create an alias for existing data types, improving code readability.

typedef unsigned long int ULONG;
ULONG distance; // Equivalent to unsigned long int distance
